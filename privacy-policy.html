Privacy Policy

Effective date: April 22, 2025

This Privacy Policy describes how the MemePoster_Bot (“the Bot”) on Telegram handles user data.

1. Data Collection

The Bot collects only the following data:

  • Telegram chat ID – to know where to send responses
  • Telegram user ID – to support bot features like user preferences or moderation

This data is part of the public Telegram API and is necessary for the bot to function correctly. No personal identifying information (PII) like names, phone numbers, or profile photos is collected.

2. Use of External APIs

The Bot may request media from external APIs (such as Pinterest) to generate meme content. These requests are not associated with specific users and are anonymous.

3. Data Storage

The Bot may temporarily store chat IDs and user IDs for session or logic purposes (e.g., to avoid spamming or customize replies), but this information is not shared, sold, or stored in external databases.

4. Security

All data processed by the Bot remains on the server and is handled with minimal retention. No third parties have access to this data.

5. Contact

If you have any questions or concerns about this policy, please contact the developer at fear9579@gmail.com.

This bot is a non-commercial, experimental project.
